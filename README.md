# Natural ou Fake Natty? Como Vencer na Era das IAs Generativas

Projeto criado para fins de estudos dando seguimento ao curso "CAIXA - IA Generativa com Microsoft Copilot". 


```markdown

##Projeto - Beirando a realidade, ou não? 

## 📒 Descrição
Criação de personagem animado com IA o mais proximo possivel da realidade.

## 🤖 Tecnologias Utilizadas
- Leonardo.Ai para criação de imagem.
- Lumalabs para aniamção da imagem
- D-ID para criação de avatar animado 

## 🧐 Processo de Criação
- Preimeiramente realizei a criação das imagens no Leonardo.Ai, utilizando uma imagem de pessoa real (minha namorada).
Utilizei a opção "Character reference" em "Image Guidance"
- Realizei a animação do primeiro video atravez no LumaLabs
- Por fim realizei a criação de um avatar animado com base na mesma personagem.

## 💭 Reflexão 
Existem diversas inteligências artificiais, cada uma com suas particularidades.
Para criar um conteúdo próximo ao "natural", é necessário praticar e tentar várias vezes até alcançar um bom resultado.
Porém, não é nada de outro mundo e, principalmente para quem gosta, é um verdadeiro parque de diversões.
O único problema atualmente é que todas têm preços em dólar para contratação.
